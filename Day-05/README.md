## 🔌 Networking: The Switch (Layer 2)
Beyond hardware connectivity, I explored the "Brain" of the Local Area Network (LAN).

### Key Concepts Mastered:
- **MAC Address Table:** Learned how switches map physical addresses to specific ports.
- **Micro-segmentation:** Understanding how switches send data only to the intended recipient, unlike a Hub.
- **Security Takeaway:** By isolating traffic, a switch inherently reduces the surface area for "sniffing" attacks compared to legacy network hubs.


> "Security is built on a solid foundation. You can't protect a network you can't navigate."
