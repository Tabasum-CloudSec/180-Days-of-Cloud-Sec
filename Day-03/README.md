# Day 3: Linux File Manipulation 📂

## Commands Mastered:
- `touch`: Created a new security report.
- `cp`: Cloned the file to create a backup.
- `mv`: Renamed the report for the final audit.
- `ls`: Verified the file lifecycle.

## The Security Win:
I successfully managed the lifecycle of a file and prepared it for a security audit. Every command was executed in the university lab environment.
